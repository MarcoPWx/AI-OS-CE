## Tech Stack Cheat Sheet (condensed)

- Platforms: TypeScript 5.x, React 19 (web), React Native 0.79 (Expo 53)
- UI: react-native + react-native-web, React DOM
- Data: TanStack Query, fetch/axios, AsyncStorage/localStorage
- Styling: nativewind + tailwindcss, gradients, platform tokens
- Realtime: WebSocket, socket.io-client (optional)
- Backend: Supabase (Auth, Postgres, Realtime)
- Mocks: MSW v2 (web/tests) + MockEngine (RN), MockWebSocket
- Storybook: v8, Test Runner (play()), Chromatic
- API Docs: Swagger UI, API Playground
