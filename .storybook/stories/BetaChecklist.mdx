# Beta Readiness Checklist

This page summarizes the steps needed to ship the ingestion → quality → curation → DB delivery flow for beta.

See also: docs/BETA_READINESS_CHECKLIST.md for the full list.

## Data ingestion & quality

- [ ] Import CLI packs

```bash path=null start=null
node scripts/cli-import.js
```

- [ ] Heuristic quality pass

```bash path=null start=null
node scripts/quality-evaluate.js
```

- [ ] Optional AI review (OpenAI-compatible)

```env path=null start=null
AI_REVIEW_ENABLED=true
AI_REVIEW_URL=http://localhost:11434
# Or hosted
# AI_REVIEW_URL=https://api.openai.com
# AI_REVIEW_API_KEY={{OPENAI_API_KEY}}
AI_REVIEW_MODEL=gpt-4o-mini
AI_REVIEW_THRESHOLD=0.6
AI_REVIEW_MAX=100
AI_REVIEW_BATCH=10
```

- [ ] Curate for editors

```bash path=null start=null
node scripts/curate-quality.js
```

Outputs: data/cli_import/curated.json, curated.csv

## Database & API

- [ ] Generate seed and apply in Supabase (003_question_delivery)

```bash path=null start=null
node scripts/generate-sql-from-import.js
# Apply supabase/cli_import_seed.sql in Supabase
```

- [ ] Switch API to DB mode

```env path=null start=null
SUPABASE_URL=... # your URL
SUPABASE_SERVICE_ROLE_KEY={{service_role_key}}
API_USE_SUPABASE=true
```

- [ ] Smoke test

```bash path=null start=null
curl "http://localhost:PORT/api/quiz/questions?limit=5&random=true"
```
